- 모드: 단변량 (univariate), 다변량 (multivariate)
- 사용한 예측 길이 (prediction lengths): 24, 48, 168, 336, 720, 960

자세한 사용법과 설정은 [TS/Time-series-forecasting](TS/Time-series-forecasting/README.md)에서 확인하세요.
